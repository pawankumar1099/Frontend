# JavaScript Assignment Tasks
## Description
Collection of small JavaScript assignments used for practice and evaluation. Each assignment is self-contained and demonstrates a focused concept (DOM, events).

## Folder structure (recommended)
- /AsssignmentTasks
    - /Task-Name
        - index.html
        - script.js
        - styles.css 
        - /assets (images, data)
    - /Task-02-Name
    - ...
    - README.md
